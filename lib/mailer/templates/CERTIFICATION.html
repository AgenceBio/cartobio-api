Votre parcellaire a été certifié dans CartoBio. Votre attestation de production est disponible sous trois formats :

  • L'attestation de production PAC listant les parcelles déclarées à la PAC certifiées avec le détail par parcelle.
  • Un fichier ZIP contenant l'attestation de production PAC détaillée ainsi qu'un document allégé avec la liste des parcelles uniquement.
  • La liste complète des parcelles de l'exploitation.

Vous pouvez télécharger votre attestation de production sur la page de votre parcellaire certifié, via le bouton "Attestation" : {{version_name}}

La génération du pdf peut prendre un peu de temps. Vous le retrouverez ensuite sur votre ordinateur, dans les fichiers téléchargés.
